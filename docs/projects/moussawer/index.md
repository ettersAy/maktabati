# Moussawer

Client & Photographer booking platform.

## Overview

Moussawer connects clients with photographers for event bookings.

## Tech Stack

| Layer | Technology | Version |
|-------|------------|---------|
| Backend | Laravel | 13.x |
| Frontend | Vue 3 | 3.5.x |
| Database | MySQL | 8.4 |
| DevOps | Docker/Sail | Latest |
